更新及卸载 ​

更新 ​

安装ncu ​

注意

已经安装过的可以无视

由于直接用命令更新，不能改变 package.json 的内容

我更推荐用 npm-check-updates，即 ncu

sh
npm i -g npm-check-updates
1

ncu可以可以检查出需更新的软件包

sh
ncu
1

更新 ​

更新模块包版本信息

说明

这时候我们的 package.json 里的版本都按最新的填写完毕了

sh
#即运行ncu -u to upgrade package.json
ncu -u
1
2

我们直接用命令更新安装依赖即可

sh
pnpm install
1
sh
yarn install
1
sh
bun install
1

卸载 ​

卸载就比较简单了，在 package.json 中找到包名即可

sh
pnpm uninstall <包名>
1
sh
yarn uninstall <包名>
1
sh
bun uninstall <包名>
1

比如

pnpm uninstall vue

vue 就是包名
