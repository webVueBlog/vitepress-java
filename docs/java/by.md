> float 与 double

Java 不能隐式执行向下转型，因为这会使得精度降低。

1.1 字面量属于 double 类型，不能直接将 1.1 直接赋值给 float 变量，因为这是向下
转型。

`// float f = 1.1;`

1.1f 字面量才是 float 类型。
